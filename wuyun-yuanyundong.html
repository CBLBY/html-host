壹圆运动 · 核心模型

彭子益《圆运动的古中医学》的核心命题：人体即一小宇宙。宇宙大气的圆运动产生五行，人秉大气五行而生脏腑。人体内五行的流动，不是静态的配属关系，而是 一个完整的、永不停息的圆运动。

⚡

关键洞见：五运六气描述的是天道的圆运动（天干化运、地支化气），而《圆运动的古中医学》描述的是人道的圆运动（脏腑升降、经络流注）。两者是同一圆运动在不同尺度上的投影。

五行圆运动模型 — 中气（土）居中央为轴，木升、火浮、金降、水沉为四维之轮

▎中气如轴，四维如轮

轴运轮行：脾胃中气正常旋转，肝木才能从左升，肺金才能从右降，心火才能浮于上，肾水才能沉于下。

轮运轴灵：四维升降通畅，中气自然充盈。两者互为因果，构成人体生命活动的动态平衡。

此模型直接继承黄元御"一气周流"思想，而彭子益以河图为框架将其体系化。人体内五行流动，本质就是这副圆运动图在每一瞬间的运转状态。

贰五行 · 升降浮沉

彭子益将五行的本质定义为 大气圆运动的五种物质形态。人体内五行流动规律，对应着大自然一年的气机变化：春升、夏浮、秋降、冬沉，土运于中。

相火：圆运动的枢纽

彭子益特别强调 "五行各一，惟火有二"。君火主升、相火主降。君火由相火转化而来，相火又必须通过秋金收敛降入水中。这个"火降水中"的过程，是圆运动最关键的环节——它完成了能量从天上到地下、从浮到沉的循环。

⟳

相火下降的过程： 夏时太阳热力（相火）→ 秋金收敛 → 降入土下 → 藏于水中 → 来年春季升出为木气（君火之根）。人体中，胆经相火右降入肾水，肾水温升化肝木，肝木左升生心火。这就是人体内能量流动的闭环。

五行相生相克：圆运动的自我维持

🌱

相生 · 轮转次序

木生火（肝血温升化心火）→ 火生土（心火下煦生脾土）→ 土生金（脾精上输养肺金）→ 金生水（肺气肃降水归肾）→ 水生木（肾水温升养肝木）。此为圆运动先后之序。

⚖️

相克 · 制衡平衡

金克木（肺降制肝升太过）→ 火克金（心宣通制肺收敛太过）→ 水克火（肾封藏制心宣通太过）→ 土克水（脾运化制肾封藏太过）→ 木克土（肝疏泄制脾运化不及）。此为圆运动对待之衡。

🌀

相生相克本质

"相生者，补其不足。相克者，制其太过。相生相克，皆圆运动自身维持自身运动之圆而已。"——彭子益。五行之间没有真正的对抗，只有圆运动内部的 动态平衡。

叁脏腑 · 升降出入

人秉大气的五行之气而生脏腑。彭子益指出：肝胆的体质均在右，但肝经的作用在左（升），胆经的作用在右（降）。这一看似矛盾的论述，揭示了一个核心规律：脏腑的位置不等于功能运动方向。

人体脏腑五行升降简图 — 左升右降，中气运转

▎人体圆运动的脏腑分工

左升路线（木→火）： 肾水温升 → 肝木疏泄升发 → 心火宣通浮散

右降路线（金→水）： 肺金收敛 → 胆经相火右降 → 胃气下降 → 肾水封藏

中枢（土）： 脾升胃降，运化水谷精微，为四维升降提供物质基础。"中气如轴，四维如轮"，轴不转则轮不运。

脏腑升降的经典辨证

黄元御圆圈论：脾升→胃降→肝升→胆降→心火下煦→肺金右降→肾水温升。任何一个环节阻滞（如"胃气不降""胆火不降""肝气不升"），整个圆运动就会失圆，疾病由此而生。

人体内五行流动的完整闭环

肾水封藏 → 水温上升 → 肝木疏泄 → 木气升发 → 心火宣通 → 火浮于上 → 火降（相火）→ 肺金收敛 → 金气右降 → 肾水封藏

↑ 中气（脾胃）旋转于中央，驱动此轮 ∞

十二经流注规律：每一对脏腑的经络，都构成一个小圆运动——脏经自下而上（升），腑经自上而下（降）。十二条经串联起来，在人体形成一个 头→手→胸→足→头 的大圆运动。五运六气的外在周期变化，通过这个大圆与人体气机耦合。

伍五运六气 · 天人气交

五运六气是天地大气的圆运动在 年周期尺度 上的表达，而人体内的五行圆运动是同一原理在 个体身内尺度 上的表达。两者通过"气交"实现耦合。

▎五运太过不及对人体圆运动的影响

岁运太过：某一行气偏盛，打破圆运动平衡，导致"所胜受抑、所不胜反侮"。例如木运太过→肝气横逆→克脾土（木克土）→脾升胃降失灵→中轴卡顿。

岁运不及：某一行气不足，所不胜之气来乘。例如木运不及→金气来克（金克木）→肝气不升→左升路线受阻→心火不旺→整个圆运动偏向右降而左升不足。

此即五运六气作用于人体内五行流动的核心机制——改变圆运动的振幅和相位。

陆病理 · 圆运动失圆

治疗本质："中医的医理，只是运动轴的旋转，去运动轮的升降，与运动轮的升降，来运动轴的旋转而已。"——彭子益。不是对抗疾病，而是 恢复圆运动。轴不转则运轴，轮不运则转轮。此即中医治病的底层逻辑。

五运六气致病的圆运动解释

以2026丙午年为例（水运太过，少阴君火司天，阳明燥金在泉）：

水运太过 → 肾水封藏过度，水寒不温→肝木升发受抑（水寒木郁）→左升路线受阻→心火得不到肝气温煦→火气宣通不足。

少阴君火司天 → 上半年火气偏盛，火克金→肺金收敛不及→右降路线减弱→相火不降郁于上→形成"上热中湿下寒"的圆运动分离。

核心矛盾：水运太过（下寒）与司天君火（上热）之间的圆运动断裂，水火不交，中气受累。

彭子益的《圆运动的古中医学》提供了一个优雅的框架 —— 将《内经》五运六气从"天"拉回"人"身，告诉我们：天道造化的圆运动，在人身上就是脾升胃降、肝升肺降、心火下煦、肾水温升。运化失圆则病，复圆则康。
